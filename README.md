# Ampere

A bittorrent client that runs in the browser without plugins.

## Tech

Built using clojurescript (that subsequently compiles down to javascript) and utilizing the latest in HTML5 tech
* WebRTC
* HTML5 Filesystem
* Websockets
* Typed Arrays

## Usage

```bash
lein deps
lein git-deps
lein run
```
